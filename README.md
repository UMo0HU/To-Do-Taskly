## Overview

### The App

Users should be able to:

- Do CRUDS Operations on tasks:
- Create: Add New Tasks To The List.
- Read: Show All Tasks.
- Update: Edit Any Task Name.
- Delete: Delete Any Task.
- Store: Save Task List in LocalStorage.
